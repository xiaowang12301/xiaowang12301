### 👋 关于我

通信工程本科在读，主攻嵌入式系统开发、FPGA 设计与硬件电路设计。具备从原理图绘制、PCB Layout 到嵌入式软件驱动的完整项目开发经验。在电子设计竞赛、嵌入式设计大赛、FPGA 应用设计大赛中多次获得国家级奖项。✨

- ⚙️ **嵌入式方向**：使用 HAL/LL 库编程，可以使用 FreeRTOS 完成简单的多任务开发
- 🔌 **FPGA 方向**：会用 Verilog 完成初级的信号处理，具备图像处理（边缘检测、颜色识别、形状识别）FPGA 实现经验
- 🛠️ **硬件设计**：使用立创 EDA，独立完成四层板设计，掌握 ADC/DAC 采集电路、运放电路、滤波电路设计及高速信号完整性
- 📟 **未来展望**：从通信er变为计算机体系架构er
- 💬 常聊的话题：STM32、FPGA、FreeRTOS、立创 EDA、电子竞赛、硬件调试、信号处理，ysyx

  - 展现较强的电路设计与系统调试能力

- 🏅 **海云捷讯杯 FPGA 应用设计大赛 省级奖项** — *2024.09 - 2025.08*
  - 基于 FPGA 开发物块视觉识别系统，实现颜色识别、形状识别、旋转角度检测
  - 采用流水线架构实现 30fps 实时图像处理，通过 UART 与机械臂控制板通信

### 🚀 代表项目

- 🦾 **FPGA 视觉识别与机械臂控制系统** — *FPGA 模块负责人*
  - 基于 Xilinx FPGA 开发视觉识别模块，实现物块颜色/形状/旋转角度实时识别
  - 设计图像处理流水线：采集 → 预处理 → 特征提取 → 结果输出
  - 实现 Sobel 边缘检测、颜色阈值分割、轮廓提取等算法的硬件加速
  - 双 FPGA 架构协作，系统处理延迟 < 50ms，识别准确率 > 95%

- 🌿 **智慧农林环境监测系统** — *系统架构与软件负责人*
  - 基于瑞萨 RA 系列 MCU，荣获嵌入式设计大赛国家级三等奖
  - 架构：MCU 主控 + 5 路传感器（PM2.5/温湿度/土壤湿度/烟雾）+ LCD + 电源管理
  - 使用 FreeRTOS 实现多任务实时系统，完成数据曲线绘制与报警阈值设置

- ⚡ **电磁循迹小车系统设计** — *硬件负责人*
  - 基于 STM32F103 设计完整硬件电路与 PCB，实现速度环+角度环串联 PID 控制
  - 完成直线加速、弯道跟踪、进环逻辑等关键功能开发

- 📟 **高精度信号测量装置设计** — *主要开发者*
  - 针对电赛仪器仪表类题目设计高精度信号采集与处理系统
  - 掌握锁相放大、相关检测、斩波稳零等微小信号检测技术
